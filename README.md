## Deskripsi Program
Program CLI untuk membandingkan performa 6 algoritma sorting:
- **Sorting Dasar** (1000 data integer random): Bubble Sort, Insertion Sort, Selection Sort
- **Advance Sorting** (dataset kata): Merge Sort, Quick Sort, Shell Sort

## Fitur Program
- Generate 1000 data integer random otomatis
- Menggunakan data kata dummy (tidak perlu download file eksternal)
- Tampilkan data asli, data setelah shuffle, dan data setelah sorting (10 data pertama)
- Shuffle data sebelum sorting
- Hitung waktu eksekusi menggunakan clock() dalam milidetik
- Sorting secara ascending
- Tanpa fungsi sorting bawaan
